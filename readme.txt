Profile Update Notifications for User Profiles Made Easy.

== Description ==
[Profile Update Notifications for User Profiles Made Easy](https://goo.gl/BfgMuf) is a free addon provided by UPME for sending email to admins about notifications on updates of certain fields inside UPME profile.

Please note that this addon only works for UPME frontend profile updates and doesnâ€™t work for backend profile updates.

Please note that this is an addon for  [User Profiles Made Easy](https://goo.gl/w716O3) plugin and you need at least UPME version 2.2.03 to use it.

> **Main Features**
>
> * Email notifications on custom field updates.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/upme-profile-notifications` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the docs at https://goo.gl/BfgMuf to use profile update notifications.
